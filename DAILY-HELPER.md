# Daily Copy-Paste Helper (No Thinking Required)

If you're here, you're doing it right.

## Every Time (Same Steps)
1. Open this repo
2. Click Code → Codespaces → Create codespace on main
3. Open the terminal (bottom black window)
4. Paste ONE block from ChatGPT
5. Press Enter
6. Go back to GitHub
7. Click Compare & pull request
8. Click Create pull request
9. Click Merge pull request
10. Delete branch + delete Codespace

That’s it. Nothing else.

---

## Rules
- There is no Save button
- Terminal runs instantly
- If it worked, GitHub shows it
- Green check = good
- Red X = ask for help

You are not expected to remember anything.
